# File-Type Digging Project

## Introduction

本项目为个人用于探索并且学习各种不同类型的文件格式的原始二进制存储方式的学习项目。我会尝试使用 Rust 根据规范还原各种不同格式的基础实现。

## Restrictions

- 由于出于学习而不是实用目的，所以只会实现主流程，分支流程如果意义不大会直接略过。
- 由于不会进行完整实现，代码可能出现很多边界 Case 与 Bug，如果不是流程错误则不会进行修改。
- 如果一种格式随着时间更新出现了多个版本的实现，只会选取最流行的版本进行实现。如果存在多个流行版本且互相之间的实现并不兼容，则会考虑都尝试实现一遍。
- 我会尽量详细的补充注释与 README。
- 由于是个人项目并且我的英文并不好，大部分文本内容会使用中文撰写。
- 部分实现会借助 Claude 与 GPT。
- 尽量不会出现规范不开源的格式，有法律风险且麻烦，依赖逆向工程并不靠谱。
